# Airline Industry Analytics Case Study

## SQL + Python | Relational Database Analysis | Business Performance Insights

This repository presents a structured analytics case study built using data from a multi-table airline industry database. The objective of the project was to evaluate airline operational performance, flight volume trends, delay patterns, and weather relationships using a combination of SQL and Python.
The dataset consists of several relational tables, including flights, airlines, airports, planes, and weather. SQL was used to perform aggregations, joins, filtering, and performance calculations across these tables. Python (via pandas) was then used to execute SQL queries programmatically, structure outputs into reusable DataFrames, and generate visual analysis for interpretation.

### Key analytical components include:

- Counting and validating records across database tables

- Identifying on-time departure and arrival records

- Calculating mean arrival delays across airlines

- Determining flight volume by carrier and month

- Building reusable functions to dynamically query monthly flight counts

- Comparing peak operational periods across carriers

- Analyzing the relationship between temperature and wind speed

- Ranking airlines by average arrival delay and visualizing performance differences

This project demonstrates the integration of relational database querying with Python-based analytical workflows. Beyond writing static SQL queries, the analysis emphasizes reproducibility through modular functions and structured outputs that can be extended into dashboards or reporting tools.

### The work highlights practical analytics skills relevant to business and operations teams, including:

- Multi-table joins and aggregation logic

- Performance benchmarking

- Seasonal trend analysis

- Delay and efficiency measurement

- Programmatic database querying

Originally completed as a structured SQL and Python exercise, this repository reframes the work as a documented case study focused on analytical clarity, reproducibility, and real-world application.
